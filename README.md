# MAISON48 — Internal Tools

MarTech Agency setup documentation and rate card. Internal reference for founders and UTCC stakeholders.

## Files

| File | Description |
|------|-------------|
| `maison48-setup-guide.html` | คู่มือจัดตั้งบริษัท — โครงสร้าง, กฎหมาย, ทีม, การเงิน, checklist |
| `maison48-rate-card.html` | ราคาตลาด MarTech ไทย — consulting, training, stack, quote calculator |

## Usage

Open directly in any browser — no build step, no dependencies beyond Google Fonts.

```
open maison48-setup-guide.html
open maison48-rate-card.html
```

Or serve via GitHub Pages: `Settings → Pages → Deploy from main branch → / (root)`
